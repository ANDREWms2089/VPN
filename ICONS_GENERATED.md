## Чтобы увидеть новую иконку:

### Для Android:
1. **Полностью удалите приложение** с устройства/эмулятора
2. **Очистите кэш сборки:**
   ```bash
   flutter clean
   ```
3. **Пересоберите приложение:**
   ```bash
   flutter build apk
   # или
   flutter run
   ```

### Для iOS:
1. **Очистите кэш:**
   ```bash
   flutter clean
   cd ios
   pod deintegrate
   pod install
   cd ..
   ```
2. **Пересоберите:**
   ```bash
   flutter build ios
   # или
   flutter run
   ```
